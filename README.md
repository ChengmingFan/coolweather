# Fantasy weather  
  安卓练习项目  
  
  范特西天气是一款基于Android端开源的天气预报软件，具备查看全国的省市县、查询任意城市天气、自由切换城市、手动更新天气、后台自动更新天气等功能。
## 功能介绍  
* 天气信息展示（当前天气情况，空气质量，生活建议，一周七天概况）  
  
* 缓存数据，减少网络请求，保证离线查看
## 公开API
天气数据来源于：和风天气

城市信息来源于：CSDN

app背景图片： 必应每日壁纸
